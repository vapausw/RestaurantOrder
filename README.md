# 项目简介
**项目名称：快餐店点餐**

**本项目是一个快餐店点餐网站开发，主要包含两个大的功能：(1)门店扫码点餐, (2)线上小程序下单门店自提**
****
**本项目主要灵感来自于类似于蜜雪冰城，肯德基，麦当劳等快餐店的微信小程序。**
